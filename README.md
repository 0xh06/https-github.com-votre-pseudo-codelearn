# CodeLearn Pro

Plateforme EdTech de haute performance pour l'apprentissage des algorithmes et du code.

## Déploiement sur Vercel

Le projet est configuré pour un déploiement immédiat sur Vercel :

1. Connectez votre dépôt GitHub à Vercel.
2. Vercel détectera automatiquement la configuration Vite.
3. La redirection SPA est gérée par le fichier `vercel.json` inclus.

## Stack Technique
- React 19
- Vite 8 (Rolldown)
- TypeScript
- Tailwind CSS 4
- Zustand
- Framer Motion
